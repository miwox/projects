# Pong with rotating paddles

> Miwand Baraksaie

# Controls
`ESC` Exit  
`F1` Toggles Left KI  
`F2` Toggles Right KI  
`F3` Enables debug drawing
`P` Pauses the game

## Left player (WASD)
`W` Moves paddle up  
`S` Moves paddle down  
`D` Rotates paddle clockwise  
`A` Rotates paddle counter-clockwise
